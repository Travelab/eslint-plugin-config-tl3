# eslint-plugin-tl3

Code styleguide for TL3

## Installation

### Install plugin

```bash
cd my-project-dir
yarn add eslint https://github.com/Travelab/eslint-plugin-tl3.git
```

### Use it

Add these to package.json
```json
"eslintConfig": {
  "extends": "tl3"
}
```

or add these to .eslintrc.json

```json
{
  "extends": "tl3"
}
```
